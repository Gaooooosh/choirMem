# 富文本编辑器使用指南

## 功能概览

我们的富文本编辑器现在支持以下功能：

### 文本格式化
- **粗体** (Ctrl+B)
- *斜体* (Ctrl+I)
- <u>下划线</u> (Ctrl+U)
- ~~删除线~~

### 段落格式
- **段落** - 普通文本段落，可用于取消标题格式
- **标题 1** - 最大的标题
- **标题 2** - 中等标题
- **标题 3** - 小标题

### 列表
- **无序列表** - 项目符号列表
- **有序列表** - 数字编号列表

### 引用
- **引用块** - 用于引用文本或重要内容

### 媒体插入
- **链接** - 插入或移除超链接
- **图片** - 插入图片（支持本地文件上传）
- **视频** - 插入视频，支持：
  - YouTube 视频链接
  - Bilibili 视频链接
  - 直接视频文件 (.mp4, .webm, .ogg)
  - 其他iframe嵌入内容

### 编辑操作
- **撤销** (Ctrl+Z) - 撤销上一步操作
- **重做** (Ctrl+Y) - 重做已撤销的操作

## 使用技巧

### 取消标题格式
1. 选中标题文本
2. 点击工具栏中的"段落"按钮（T图标）
3. 标题将转换为普通段落

### 插入链接
1. 选中要添加链接的文本
2. 点击链接按钮
3. 输入链接地址
4. 再次点击链接按钮可以移除链接

### 插入图片
1. 将光标放在要插入图片的位置
2. 点击图片按钮
3. 在弹出的上传对话框中：
   - **拖拽上传**: 直接将图片文件拖拽到上传区域
   - **点击上传**: 点击上传区域选择本地图片文件
4. 支持的格式：JPG、PNG、GIF、WebP
5. 图片将自动上传到服务器并插入编辑器
6. 图片会自动调整宽度适应编辑器

### 插入视频
1. 将光标放在要插入视频的位置
2. 点击视频按钮
3. 输入视频链接：
   - YouTube: 直接粘贴视频页面链接
   - Bilibili: 直接粘贴视频页面链接
   - 其他: 输入嵌入代码或直接视频文件链接

### Markdown 快捷键
编辑器还支持 Markdown 快捷输入：
- `# ` - 创建标题 1
- `## ` - 创建标题 2
- `### ` - 创建标题 3
- `- ` - 创建无序列表
- `1. ` - 创建有序列表
- `> ` - 创建引用块

## 注意事项

- 插入的图片和视频需要确保URL地址有效且可访问
- 视频嵌入可能受到目标网站的跨域政策限制
- 建议使用高质量的图片和视频资源以获得最佳显示效果
- 编辑器会自动保存内容，但建议定期手动保存文章

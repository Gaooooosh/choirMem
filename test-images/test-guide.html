图片上传格式兼容性测试

支持的格式

  • JPEG (.jpg, .jpeg)
  • PNG (.png)
  • GIF (.gif)
  • WebP (.webp)
  • SVG (.svg)

测试步骤

  1. 访问 图片上传测试页面
  2. 分别测试上传不同格式的图片
  3. 检查上传状态和错误处理
  4. 验证图片在编辑器中的显示效果

测试文件

在当前目录中已创建以下测试文件：

  • test-image.svg - SVG 格式测试图片

你可以从网上下载其他格式的测试图片，或使用现有的图片文件。

预期结果

  • ✅ 所有支持的格式都能成功上传
  • ✅ 上传过程有进度显示
  • ✅ 图片能在编辑器中正确显示
  • ✅ 错误格式会显示明确的错误信息
  • ✅ 文件大小超限会被拒绝
